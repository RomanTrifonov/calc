# Арифметика калькулятора

## Обычный кредит

Месячная ставка:

```text
i = annualRate / 100 / 12
```

Аннуитетный платеж:

```text
payment = amount * i * (1 + i)^months / ((1 + i)^months - 1)
```

Если ставка равна 0:

```text
payment = amount / months
```

Помесячный расчет:

```text
interest = balance * i
principal = payment - interest
newBalance = balance - principal
```

В последнем месяце платеж корректируется по фактическому остатку, чтобы не появлялся отрицательный долг из-за округления.

## Кредитная карта

У кредитной карты нет заданного срока. Срок рассчитывается моделированием по месяцам.

```text
interest = balance * monthlyRate
principal = monthlyPayment - interest
balance = balance - principal
```

Если платеж меньше или равен процентам за первый месяц, долг не закрывается. В этом случае калькулятор должен показывать предупреждение.

## Что пока не учитывается

- страховки;
- комиссии;
- льготный период;
- изменение ставки;
- дата платежа внутри месяца;
- штрафы;
- досрочные платежи сверх указанного платежа.
