# План размещения и монетизации

## Этап 1. Техническая стабильность

1. Проверить арифметику.
2. Проверить адаптивность на iPhone, Android, планшете и компьютере.
3. Проверить PDF.
4. Завести журнал ошибок.
5. Подключить аналитику.

## Этап 2. Публичное размещение

1. Оставить GitHub Pages для теста.
2. Купить домен.
3. Подключить домен к GitHub Pages.
4. Добавить favicon и нормальное название сайта.
5. Добавить страницу с описанием методики расчета.

## Этап 3. Трафик

1. SEO-страницы под запросы: кредитный калькулятор, расчет кредитки, сравнение кредитов.
2. Короткие инструкции для Telegram и VK.
3. Видео-демонстрации.
4. Посты с примерами расчетов.

## Этап 4. Монетизация

1. Реклама банковских и финансовых сервисов.
2. Партнерские ссылки на кредитные продукты.
3. Лид-форма на подбор кредита.
4. Платная расширенная версия для консультантов.
5. White-label версия для юристов, финансовых консультантов и брокеров.

## Важное условие

Монетизация не должна ломать доверие. Сначала калькулятор должен быть точным, полезным и понятным.
