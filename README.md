# Кредитный калькулятор

Статический веб-калькулятор для расчета кредита, кредитной карты и сравнения двух вариантов задолженности.

## Функции

1. Расчет обычного кредита: сумма, ставка, срок, ежемесячный платеж, общая выплата, переплата, график.
2. Расчет кредитной карты: долг, ставка, ежемесячный платеж, срок закрытия, переплата, график.
3. Сравнение двух вариантов: кредит против кредита, карта против карты, кредит против карты.
4. Название каждого варианта для понятного PDF-отчета.
5. Цветовое сравнение ставки, платежа, срока и переплаты.
6. Сохранение результата в PDF через системную печать браузера.

## Требования к проекту

Калькулятор должен быть стабильным, арифметически точным, логичным, понятным, минималистичным и адаптивным для телефона, планшета и компьютера.

## Текущий формат

Проект работает как один статический HTML-файл без сервера и базы данных. Это упрощает публикацию через GitHub Pages и снижает стоимость поддержки.
